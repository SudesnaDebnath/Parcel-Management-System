# Parcel-Management-System
This project is a console-based Parcel Management System designed from both customer and officer perspectives.

## Project Description

This project is a console-based Parcel Management System designed from both customer and officer perspectives.

**Customer Module:**

* Registration and Login
* Parcel Booking
* Payment
* Parcel Tracking
* Booking Cancellation
* Feedback Submission

**Officer Module:**

* View Bookings
* Update Parcel Status
* Manage Delivery Workflow
* Booking Cancellation
* View Customer Feedback

The project focuses on backend logic, workflow management, validations, and JSON-based data persistence using Python.
